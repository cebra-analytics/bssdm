# Cleaned global occurrences of *Hieracium pilosella* (hawkweed)

Species occurrence coordinates (WGS84) used in package examples. Records
were downloaded from GBIF and cleaned with *CoordinateCleaner* (Zizka et
al., 2019).

## Usage

``` r
hawkweed
```

## Format

A data frame with columns:

- lon:

  Longitude (degrees).

- lat:

  Latitude (degrees).

- verified:

  Numeric flag from the cleaning workflow.
